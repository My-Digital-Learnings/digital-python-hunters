## Method Syntax 

### <ins>Method without input </ins>

    class FirstClass : 
	    def method():  
		    print("Hello - World ")
		method()
	 pass 

###### output :    `Hello - World`

### <ins>Method with input </ins>


    class FirstClass : 
	    def method(userInput):  
		    print(userInput)
		
		method("Hello- World")
	 pass 

 ###### output :    `Hello - World`
